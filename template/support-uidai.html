How to install and initialize RDService for IriShield-USB MK2120UL device?

Here is how to install RDService for IriShield-USB MK2120UL:

  • Step 1: Check RDService validity for the device here
    • If your device is not valid for RDService, please contact the seller for RDService validity.
  • Step 2: If the device is valid for RDService, let's install RDService and RDService extension packages (please choose the suitable OS)
    • Windows (RDService) (RDService extension)
    • Linux32 (RDService) (RDService extension)
    • Linux64 (RDService) (RDService extension)
    • Android (RDService) (RDService extension)
  • Step 3: Open the "IriShield USB MK2120UL Client Management" application and check if "Init" function is successful.
  • Step 4: Check if the MK2120UL was installed RDService successfully here
